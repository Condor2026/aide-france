## 🇪🇸 Español

### 1. PROPÓSITO DE LA HERRAMIENTA

**AIDE FRANCE** es una herramienta OSINT de código abierto diseñada para analizar noticias públicas de periódicos franceses y asociaciones, y detectar **necesidades sociales** (logement, emploi, alimentation, santé, précarité, droits, urgence sociale, solidarité).

**Finalidades legítimas:**

| Finalidad | Descripción |
|-----------|-------------|
| 📰 **Periodismo social** | Análisis de tendencias de precariedad |
| 🔬 **Estudios sociológicos** | Investigación sobre pobreza y exclusión |
| 🏘️ **Asociaciones locales** | Identificación de zonas con necesidades |
| 🛡️ **Ayuda comunitaria** | Información para voluntarios y trabajadores sociales |
| 📊 **Políticas públicas** | Datos agregados para mejorar la ayuda social |
| 🤝 **Solidaridad ciudadana** | Concienciación sobre situaciones difíciles |

**Lo que NO es:**

| Prohibición | Explicación |
|-------------|-------------|
| ❌ Vigilancia masiva | No monitoriza individuos específicos |
| ❌ Perfilación social | No clasifica ni etiqueta personas |
| ❌ Discriminación | No reemplaza a servicios sociales oficiales |
| ❌ Inteligencia policial | No es una herramienta de fuerzas del orden |

---

### 2. RESPONSABILIDAD DEL USUARIO

El usuario es el **ÚNICO responsable** del uso que dé a esta herramienta.

**Usos PROHIBIDOS:**

| Categoría | Ejemplos |
|-----------|----------|
| **Discriminación** | Excluir personas por código postal, situación social, origen |
| **Acoso** | Hostigar a beneficiarios de ayudas sociales |
| **Estigmatización** | Señalar a colectivos vulnerables |
| **Violación de privacidad** | Intentar identificar individuos a partir de datos agregados |
| **Fraude** | Usar la herramienta para simular necesidades falsas |

⚠️ **Nota importante:**
Los datos provienen de fuentes públicas (periódicos y asociaciones). El usuario debe verificar la información antes de tomar cualquier decisión basada en ella.

---

### 3. PROTECCIÓN DE DATOS

**AIDE FRANCE NO almacena ni procesa datos personales** (nombres, direcciones, documentos, IPs, cookies, etc.).

**¿Qué se guarda?** (solo en local, en tu máquina)

| Campo | Descripción |
|-------|-------------|
| 📰 **Título del aviso** | Texto público de la noticia |
| 📅 **Fecha de publicación** | Cuándo se publicó |
| 🏴 **Departamento** | Región geográfica (ej: Paris, Bouches-du-Rhône) |
| 🤝 **Tipo de ayuda** | Clasificación automática (logement, emploi, etc.) |
| 📰 **Fuente** | Nombre del periódico o asociación |

**Lo que NO se guarda:**

| Prohibido | Explicación |
|-----------|-------------|
| ❌ Nombres de personas | No se extraen nombres propios |
| ❌ Direcciones | No se guardan direcciones específicas |
| ❌ Teléfonos | No se almacenan números |
| ❌ IPs | No se registran direcciones IP |
| ❌ Cookies | No se utilizan cookies |
| ❌ Geolocalización precisa | Solo nivel departamental |
| ❌ Información identificable | Cero PII (Personally Identifiable Information) |

**📜 Legislación aplicable (Francia / UE):**

| Normativa | Cumplimiento |
|-----------|--------------|
| **RGPD (UE) 2016/679** | ✅ Fuera de ámbito (no trata datos personales) |
| **Loi Informatique et Libertés (1978)** | ✅ No aplica (datos anónimos agregados) |
| **CNIL** | ✅ Sin necesidad de declaración (uso local) |

Esta herramienta está **FUERA del ámbito de aplicación del RGPD** porque:
- No trata datos personales
- El usuario ejecuta el software localmente
- No hay transferencia a servidores externos
- No hay perfilación automática de individuos
